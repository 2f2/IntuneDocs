Microsoft Intune

Microsoft Intune provides mobile device management, mobile application management, and PC management capabilities from the cloud. Intune is a key component of Microsoft’s Enterprise Mobility strategy, which enables users to be productive on the devices and applications they love, while helping organizations ensure their corporate assets are secure.

The documentation on this site provides everything you need to deploy and manage Intune from the cloud. If you’re a System Center Configuration Manager customer interested in Intune, you can learn more about hybrid mobile device management in the Configuration Manager documentation library.

New to Intune? Visit our product site.

Ready to get hands-on? Try the 30 day free trial of Microsoft Intune.
