# Fix common issues with your Windows device

Sometimes issues can occur with your Windows device that may be able to be fixed through troubleshooting. Some common issues that we've seen that you can first try to fix yourself are listed below. For more complex problems, we recommend you reach out to your IT admin through their contact information on the [Company Portal website](http://portal.manage.microsoft.com).

- [You need to update your Windows device](you-need-to-update-your-windows-device.md)
- [Your Windows version isn't yet supported](your-windows-version-isnt-yet-supported.md)
- [You need to enable Windows encryption](you-need-to-enable-windows-encryption.md)
- [Your device needs antimalware software](your-device-needs-antimalware-software.md)
- [You need to enable Secure Boot](you-need-to-enable-secure-boot-windows.md)
- [You need to enable Code Integrity](you-need-to-enable-code-integrity.md)
